### Return Policies

**Customer-Friendly Returns Increase Sales**

Create return policies that build buyer confidence:

```typescript
// Ask your AI assistant:
"Create a 60-day return policy with free return shipping"

// The AI will execute:
const returnPolicy = await mcp.useTool('account_createReturnPolicy', {
  marketplace_id: 'EBAY_US',
  name: 'Easy Returns - 60 Days Free Shipping',
  description: 'We accept returns within 60 days with free return shipping',
  return_period: {
    value: 60,
    unit: 'DAY'
  },
  returns_accepted: true,
  refund_method: 'MONEY_BACK',
  return_shipping_cost_payer: 'SELLER', // Free returns
  restocking_fee_percentage: '0'
});

console.log(`‚úÖ Return policy created: ${returnPolicy.returnPolicyId}`);
```

<Note>
**Industry Standards:**
- **30-day returns:** Minimum recommended
- **60-day returns:** Competitive standard
- **Free return shipping:** Increases conversion by 10-15%
- **No restocking fees:** Improves buyer trust
</Note>

<Note>
**eBay Managed Payments** handles sales tax collection and remittance automatically in most states. You only need to configure this for states where you have nexus.
</Note>

**Problem:** Setting up a new eBay seller account requires creating multiple policies.

```typescript
async function setupNewSellerAccount() {

      { paymentMethodType: 'CREDIT_CARD' },
      { paymentMethodType: 'PAYPAL' }

    immediate_pay: true
  });
  console.log(`‚úÖ Payment policy: ${paymentPolicy.paymentPolicyId}\n`);

  console.log('Creating fulfillment policy...');
  const fulfillmentPolicy = await mcp.useTool('account_createFulfillmentPolicy', {
```

```typescript
async function expandToInternationalMarketplaces() {

      marketplace_id: marketplace,
```

### Delete Unused Policies

```typescript
"Clean up unused policies"

async function cleanupPolicies() {
  // Get all policies
  const returnPolicies = await mcp.useTool('account_getReturnPolicies', {
    marketplace_id: 'EBAY_US'
  });

  // Get all active listings to check which policies are in use
  const offers = await mcp.useTool('inventory_getOffers', {
    marketplace_id: 'EBAY_US',
    limit: 1000
  });

  const usedPolicyIds = new Set(
    offers.offers.map(o => o.listingPolicies?.returnPolicyId).filter(Boolean)
  );

  // Delete unused policies
  for (const policy of returnPolicies.returnPolicies) {
    if (!usedPolicyIds.has(policy.returnPolicyId)) {
      await mcp.useTool('account_deleteReturnPolicy', {
        return_policy_id: policy.returnPolicyId
      });
      console.log(`üóëÔ∏è  Deleted unused policy: ${policy.name}`);
    }
  }
}
```

    - Review eBay policy guidelines for your marketplace
  </Accordion>

  <Accordion title="Cannot delete policy">
    **Cause:** Policy is currently in use by active listings.

    **Solution:**
    - Find listings using the policy
    - Update listings to use a different policy
    - Wait for listings to end naturally
    - Then delete the unused policy
  </Accordion>

  <Accordion title="Policy not applying to listings">
    **Cause:** Policy ID mismatch or marketplace mismatch.

    **Solution:**
    - Verify policy ID is correct
    - Ensure policy marketplace matches listing marketplace
    - Check that policy is not deleted
    - Re-create offer with correct policy IDs
  </Accordion>

  <Accordion title="Sales tax not collecting">
    **Cause:** Incorrect jurisdiction setup or eBay Managed Payments handling.

    **Solution:**
    - Verify jurisdiction_id (state code) is correct
    - Check if eBay is automatically handling sales tax
    - Confirm your business nexus states
    - Review eBay Managed Payments tax settings
  </Accordion>
</AccordionGroup>
